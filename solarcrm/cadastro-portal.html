🌞 Cadastro em Massa — Portal do Cliente

Esta ferramenta cria usuários no Supabase Auth e vincula cada um à sua planta no GDASH.
Use apenas uma vez para cadastrar todos os clientes. Cada cliente receberá um e-mail de confirmação.

Como funciona:
1. Busca todas as plantas do GDASH
2. Para cada planta, verifica se já existe usuário com aquele e-mail
3. Se não existir, cria o usuário com senha temporária
4. Vincula o usuário à planta na tabela portal_clientes
Os clientes podem trocar depois no portal
⚠️ Use apenas localmente — nunca publique esta chave!
